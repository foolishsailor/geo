getDestinationPoint()

Get position from bearing and distance

Use origin point, bearing and distance to calculate new position

Options

Surface Model

Type of formula to use. Spherical is less accurate but also less processor intensive.

Output Format

Input and Result

Origin

Direction and Distance Travelled

Destination

Simple Implementation

Process wide range of gps position formats into a decimal format. Accepts Object, Array or String.

Point must be in Decimal - Ex: 34.2334

Distance requires type [Km , M, NM, m] - Ex: 4,234Km
