parseDMS()

Parse and Validate Geo String

Parse a human readable or error prone GPS position like string into Decimal format and validates it is a valid value. Returns error describing where the value is incorrect

Will successfully parse:   93 07 57.9 W,   W053 07,   53°07'57.9W

Will trap errors:   193 07 57.9 W,   W053 po07,   93°07'57.9S
